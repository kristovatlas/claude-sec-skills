# claude-sec-skills
Skills that I use for Claude Code which I have reviewed for prompt injection content

## Installation

### New Machine Setup
```bash
# Clone the repo
git clone git@github.com:kristovatlas/claude-sec-skills.git
cd claude-sec-skills

# Install skills
chmod +x install.sh
./install.sh
```

### Updating Skills
```bash
cd ~/path/to/claude-skills-personal
chmod +x update.sh
./update.sh
```
